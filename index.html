Etiquetas

Las etiquetas HTML van a ayudarnos a brindar una estructura y semántica al contenido de nuestro website. Cada una tiene características y usos diferentes, aunque visualmente puedan dar un resultado similar.

Muchas etiquetas son fáciles de diferenciar como los headings o las imágenes mientras que otras como header, footer, section o div, tienen su impacto en la semántica y si sabemos un poco de inglés también podremos darnos una pequeña idea de su uso. En este curso vamos a usar las más importantes, que serán parte de tu día a día como Software Developer.

DOM

El Document Object Model es una estructura de árbol que representará todos nuestros proyectos web como si un árbol genealógico fuera: padres, hijos, hermanos con niveles infinitos. Cuando entendemos esta anidación podemos identificar dependencias, herencias en css y ver qué tan complejo es nuestro proyecto.

Semántica

La semántica le brinda sentido a cada elemento existente en un sitio web. Algunos elementos serán meramente decorativos y no significarán nada, pero otros serán títulos, listas, párrafos, etc. y deberán estar en diferente jerarquías. Alguna secciones podrán ser más relevantes que otras y esto ayudará a los motores de búsqueda como Google o Duck Duck Go a diferenciar y categorizar el contenido y será la herramienta más valiosa para estar en los primeros resultados de búsqueda SEO. También ayudará a la accesibilidad del sitio web, para que personas con capacidades diferentes puedan entender cada contenido.

Atributos

Los atributos le dan características extra a las etiquetas para complementar información y son pre programadas por el navegador. Algunos atributos funcionan en todas las etiquetas de la misma forma, como ‘class’, pero otros atributos funcionan en solo algunas etiquetas, como src.

CSS

Hojas de Estilo en Cascada (del inglés Cascading Style Sheets) o CSS es el lenguaje de estilos utilizado para describir la presentación de documentos HTML.
